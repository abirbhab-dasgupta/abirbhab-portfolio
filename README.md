# Abirbhab Dasgupta — Portfolio

Personal portfolio website for **Abirbhab Dasgupta**, a Frontend Engineer and B.Tech CSE student at Adamas University, Kolkata.

🔗 **Live:** [abirbhabdasgupta.vercel.app](https://abirbhabdasgupta.vercel.app)

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) with React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Theming:** next-themes (dark mode by default)
- **Command Palette:** kbar
- **Typography:** Geist & Inter (Google Fonts)

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Intro with animated headline |
| **Trust Ticker** | Scrolling marquee of key highlights |
| **Work** | Featured projects & case studies |
| **About** | Background, skills & experience |
| **Writing** | Blog posts & articles |
| **Contact** | Get-in-touch form & social links |

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Production build
npm run build && npm start
```
